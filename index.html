1. Use of the Spice Tune Bot

The Spice Tune bot provides music streaming functionality through Discord. By using the Spice Tune bot, you acknowledge that the bot leverages an api and its associated services to stream SoundCloud songs to your server. Additionally, the bot collects specific data as outlined in the Privacy Policy section below.

2. Data Collection

Spice Tune collects the following data from users:

  • Server IDs: This data is collected to provide the music streaming service to your server.
  • User IDs: This data is collected to associate liked songs and recommendations with specific users. Please note that Spice Tune only collects data when a user likes a song on the music player.

3. Data Usage

Spice Tune utilizes the collected data in the following manner:

  • Storing Liked Songs: When a user likes a song, the bot attempts to locate its Spotify track ID and stores it in a Firestore database. This enables the creation of custom blends between other users in your server and provides personalized recommendations based on liked songs.

4. Data Storage and Security

Spice Tune stores all collected data in an encrypted format to ensure data security. We employ appropriate measures to protect the data from unauthorized access, use, or disclosure.

5. User Rights

Users possess the right to control their personal data collected by Spice Tune. If a user wishes to delete their data, they can use the /likes command to view their liked songs and choose which ones to delete. If all songs are deleted, the user's ID will no longer be stored in our database.
